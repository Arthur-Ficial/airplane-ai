# Airplane AI -- App Store Marketing Copy

## Short Description (App Store subtitle, 30 words max)

Private AI chat that runs entirely on your Mac. No internet, no accounts, no telemetry. Powered by Gemma 4 E4B and Metal GPU acceleration. One purchase, yours forever.

## Long Description (App Store description)

Airplane AI is a private AI chat app that runs entirely on your Mac. No internet connection. No cloud servers. No accounts. No telemetry. Every conversation stays on your machine, enforced by macOS kernel-level sandboxing.

ASK ANYTHING, OFFLINE

Airplane AI ships with Gemma 4 E4B, a capable instruction-tuned language model from Google, quantized and optimized for Apple Silicon. Write emails, brainstorm ideas, debug code, draft documents, or just think out loud -- all without an internet connection. The model runs locally through llama.cpp with Metal GPU acceleration, streaming responses token-by-token as they generate.

MULTIMODAL INPUT, LOCAL PROCESSING

Paste or drag images into chat -- Apple Vision extracts text and labels on-device. Drop PDF, Word, Markdown, or code files and see the extracted text before you send. Hold the mic button to dictate with Apple Speech Recognition, fully on-device. The AI model sees only plain text. You see exactly what it sees. No hidden processing, no surprises.

BUILT FOR PRIVACY

This is not a promise. It is an engineering constraint.

Airplane AI ships with App Sandbox enabled and zero network entitlements. macOS enforces this at the kernel level -- the app physically cannot connect to the internet. There is no analytics SDK, no crash reporter, no remote config, no update checker that phones home. Four automated verification scripts run on every build to ensure no network code, no forbidden dependencies, and no unauthorized entitlements slip in.

Your conversations are stored locally in SwiftData inside the app sandbox. They are not synced to iCloud. They are not exported. They are not accessible to other apps.

DESIGNED FOR YOUR MAC

Runtime profiles are calibrated per memory class. A 16 GB MacBook Air gets a conservative context window. A 64 GB Mac Studio gets a generous one. GPU offload, batch sizes, and flash attention policies are tuned to each tier based on real benchmark data, not guesswork.

The app launches to a single, clean window. Sidebar for conversation history. Main area for chat. Composer at the bottom. Streaming Markdown with code blocks, lists, and emphasis rendered live. No tabs, no panels, no clutter.

HONEST AI

The system prompt tells the model it is offline and local. It will not pretend to browse the web, check live facts, or access your files. It will not invent citations or fabricate sources. When it does not know something, it says so.

The UI reinforces this honesty. When you attach an image or document, you see the exact extracted text that the model will receive. What you see is what the model gets.

REQUIREMENTS

- Apple Silicon Mac (M1 or later)
- macOS 15.0 (Sequoia) or later
- 16 GB unified memory minimum
- Approximately 5 GB of disk space

Note: Macs with 8 GB of memory are not supported. The model requires more memory than an 8 GB system can reliably provide for a stable experience.

ONE PURCHASE

Airplane AI is a one-time purchase. No subscriptions. No in-app purchases. No "free tier" with limits. Buy it once, use it forever, update it free.
